# Documentation

This is the main documentation for the cherrybomb application.

Documentation for the Python codebase (GUI + engine) is made using Doxygen (Python
via a filter). cherrybomb is single-language Python — no C++/TS components remain
after the rewrite, so TypeDoc/Doxygen-for-C++ are no longer used.

This documentation folder can be built and hosted as a website for sharing.
